# API V0 Documentation

V0 of the API was created exactly to the specification of the
[original dog API](https://dog.ceo/dog-api/), to make migrating as easy as
possible.

Future versions of the API will be improved on substantially.

## Paths

The base URL for all of these paths is `https://dog.jamalam.tech/api/v0/<path>`.

All successful responses, excluding `/pong`, will be in the form of a JSON
object with a field `status` that will be equal to `success`.

### `/pong`

Returns: `Pong!`, if the API is running

<details><summary>Example Response:</summary>

```json
Pong!
```

</details>
